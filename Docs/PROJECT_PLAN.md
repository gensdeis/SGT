## 추천 알고리즘 + DDA

### 파이프라인

```
STEP 1  플레이 이력 수집 (완료/이탈/재도전/SR)
STEP 2  태그 가중치 계산 (재도전 횟수 가중 2배)
STEP 3  DDA 강도 조절 (SR > 80% → 강도↑ / SR < 40% → 강도↓)
STEP 4  세션 큐 구성 (선호 태그 + DDA 강도 + 신규 발견 20%)
STEP 5  피드백 반영 (세션 종료 후 배치)
```

### Cold Start

- 초기 3세션: 오리지널 + 인기 순 + DDA 중간 강도
- 3세션 이후: 태그 + DDA 동시 적용

## 수익 모델

### 모바일 (Android/iOS)

```
무료 플레이
├── 전체 미니게임 플레이 가능
├── 글로벌/게임별 랭킹 참여
├── 세션 종료 후 전면 광고 (세션 중 광고 없음)
└── 배너 광고 상시

광고 제거 $2.99 (1회)          ← 핵심 수익원
├── 모든 광고 영구 제거
└── 구매자 전용 프로필 배지

광고 삽입 원칙
└── 세션 중 광고 없음. 결과 화면 이후에만 삽입.
```

### Steam ($7.99)

```
Steam 독점 콘텐츠
├── Steam 독점 오리지널 2종 (모바일 미출시)
├── HD 에셋팩 (4K 월페이퍼, 이모티콘)
├── 오프라인 플레이 지원
├── UGC 조기 접근 (Phase 3)
├── Steam 도전과제 / 트레이딩카드
└── 출시 세일 20% → $6.39
```

## UI/UX 설계

### 모바일 (YouTube 모바일 참고)

```
상단: 로고 + 검색바 (전 탭 공유)
하단 탭 4개:
  홈       알고리즘 세션 즉시 시작 버튼 + 취향 맞춤 카드
  인기     지금 핫한 / 역대 명작 / 클리어율 / 재도전 필터
  보관함   하트 누른 게임 + 내 최고 점수
  설정     광고제거 $2.99 카드 상단 고정 + 계정/그래픽/버그제보
게임 카드: 9:16 썸네일 · 태그 · 플레이 수 · 하트
```

### PC (Steam 클라이언트 참고)

```
좌측 사이드바:
  홈 / 인기 / 보관함 / 리더보드 / 설정
  하단: 태그 목록 빠른 필터
  최하단: 유저 정보 + 글로벌 랭킹
상단: 검색바 + "▶ 바로 시작" 버튼
리더보드 탭: 게임별·글로벌·친구·수치의전당 동시 표시
게임 카드: 동일 9:16 썸네일 · 5열 그리드
```

## UGC 저작권 & 심사 정책

### 자동 필터링 (Phase 3 오픈 전 완성 필수)

| 필터 | 감지 | 처리 |
|------|------|------|
| 저작권 이미지 | 유명 IP/로고 해시 | 자동 차단 + 크리에이터 알림 |
| 저작권 음원 | 음악 지문 매칭 | 오디오 제거 or 등록 거부 |
| 혐오 표현 | 텍스트 키워드 + 이미지 분류 | 자동 차단 + 수동 검토 큐 |
| 성인 콘텐츠 | 이미지 분류 모델 | 자동 차단 |
| 커뮤니티 신고 | 유저 신고 누적 | 임계치 초과 시 자동 비공개 |

### 심사 정책 설계 시점

```
Phase 1  정책 설계 문서화
Phase 2  파트너 베타로 오탐률 검증
Phase 3  자동 필터 전면 적용 후 UGC 오픈
```
