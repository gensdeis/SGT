🐸 개구리 잡아라
화면에 랜덤 등장하는 개구리를 빠르게 터치! 두더지 잡기(Whac-A-Mole) DNA의 반응속도 게임.
반응속도 세로 9:16 원터치 구현 완료

🎯 게임 컨셉 & 재미 요소

  • 핵심 공식: 랜덤 등장 + 빠른 반응 = 도파민 분비 (Whac-A-Mole 검증 공식)
  • 터치 성공 즉시 피드백 (+10) + 실패 패널티 (-2) → 리스크 긴장감
  • "한 번만 더" 심리: 30초라는 짧은 시간이 재도전 욕구 극대화
  • DDA: 쉬움(-1)은 1.5배 느린 등장, 어려움(+1)은 0.7배 빠른 등장

📱 인게임 UI 컨셉

  • 상단 좌측: 시간 바 (30초 감소)
  • 상단 우측: 현재 점수
  • 중앙: 9:16 잔디/연못 배경, 개구리 랜덤 위치 출현
  • 터치 시: 물튀김 파티클 + 점수 플로팅 (+10 / -2)
  • 종료: 결과 오버레이 + 최고 기록 비교

👆 조작 방식

원터치: 등장한 개구리를 탭. 빈 공간 탭 시 미스 (-2). 개구리는 1~2초 간 머물다 사라짐. 동시 2마리 이상 등장 가능 (DDA+1).

📦 필요 리소스

개구리 idle 스프라이트 개구리 caught 스프라이트 잔디/연못 배경 (9:16) 터치 이펙트 (물튀김) 포획 효과음 (뽁!) 미스 효과음 BGM (경쾌한 루프)

라면 물 게이지가 올라간다! 60~80% sweet spot 에서 정확히 터치하는 타이밍 게임. 5라운드.

🎯 게임 컨셉 & 재미 요소

  • 라면 끓이는 과정 = 한국인의 일상 밈. 즉각적인 공감대
  • sweet spot 타이밍: 60~80% 구간 정중앙 터치 시 최고점
  • 80% 초과 = 라면 넘침 = 0점 → 극적 긴장감
  • 5라운드 = 누적 긴장 + 1라운드 실패해도 복구 기회
  • DDA: 쉬움은 50~85%, 어려움은 65~78% (좁은 구간)

📱 인게임 UI

  • 중앙: 라면 냄비 일러스트 + 물 올라오는 애니메이션
  • 우측: 세로 게이지 바 (sweet spot 영역 녹색 표시)
  • 상단: 라운드 카운터 (2 / 5), 누적 점수
  • 터치 시: PERFECT / GOOD / FAIL 텍스트 팝업

👆 조작

게이지가 0%→100% 로 ~6초간 상승. sweet spot 구간에서 화면 아무 곳이나 탭. 한 라운드에 한 번만 탭 가능. 안 탭하면 100% 도달 → 넘침 = 0점.

📦 필요 리소스

라면 냄비 스프라이트 물 끓는 애니메이션 (4~8프레임) 증기 이펙트 부글부글 효과음 주방 배경 (9:16) PERFECT/FAIL 팝업 텍스트 BGM (코믹한 긴장감)
🤖 AI 프롬프트 냄비: "A boiling ramen pot sprite, top-down view, steam rising, cute cartoon style, red pot with ramen noodles inside, transparent background, 512x512, for mobile cooking mini-game"

배경: "A cozy Korean kitchen background, warm tones, wooden counter, 9:16 portrait, cartoon illustration style, soft lighting, for mobile casual game, 1080x1920"
