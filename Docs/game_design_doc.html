00 — 세계관 & 스토리

놀이짱이 되어라

아이들에게 노는 방법을 가르치고 평가하는 전무후무한 학교. 이 곳에서 놀이는 하나의 정식 과목이다.

📖 배경 스토리

공부만을 강요하던 어른들은 아이들이 놀이를 통해 건강하게 스트레스를 해소하는 것이 얼마나 중요한지 깨달았다. 그렇게 탄생한 놀이특성화초등학교. 선생님들은 아이들에게 노는 방법을 가르치고, 평가한다. 이 학교에서 제일 잘 노는 학생은 인기스타이자 짱이다.


시대의 흐름에 맞춰 "공부특성화초등학교"에서 전학 온 주인공. 학교에 있는 모든 놀이 고수들을 이기고 놀이짱이 되어야 한다!

🎯 핵심 판타지

진지한 학교 세팅에서 놀이가 공식 평가 과목이라는 역발상. 공부 대신 게임 실력으로 서열이 정해지는 세계.

🏫 진행 방식

탑뷰 RPG 스타일 오버월드(학교 맵)를 이동하며 각 장소에서 다른 미니게임 고수들과 대결. 승리 시 다음 구역 해금.

🎮 플랫폼 & 엔진

Cocos2D 기반 PC 게임. 픽셀 아트 그래픽, 실제 사진을 배경으로 활용하는 하이브리드 비주얼 스타일.

🖱️ 교실 클릭 게임
수업 시간, 몰래 마우스를 클릭하라 — 제한 시간 안에 최대한 많이!
반응속도 탑뷰 마우스 클릭

🎮 핵심 메커닉

교실 탑뷰 화면에서 선생님의 눈을 피해가며 마우스를 빠르게 클릭한다. TIME(남은 시간) / HIGH(최고 기록) / CLICK(현재 클릭 수) 세 가지 지표를 실시간 표시.

🏫 씬 구성

  • 벽돌 패턴 바닥의 교실
  • 5×5 격자로 배치된 학생 책상
  • 청색 교복 착용 캐릭터들
  • 상단 칠판 영역에 UI 배치
  • 하단 바 = 시간 게이지

📊 UI 요소

  • TIME — 남은 시간 (소수점 2자리)
  • HIGH — 세션 최고 클릭 수
  • CLICK — 현재 클릭 카운트
  • 하단 주황색 타임바 (감소)

⚙️ 게임 루프

  • 시작: 타이머 시작 (~15초)
  • 플레이: 마우스 빠르게 클릭
  • 종료: 시간 0 → 결과 표시
  • HIGH 갱신 시 하이라이트
  • 재도전 or 다음 스테이지
핵심 재미 요소: 단순하지만 중독성 강한 클릭 경쟁. "조금만 더 빨리 클릭하면..."이라는 욕구를 자극. 하이스코어 갱신 시 성취감 극대화. 교실이라는 현실적 배경이 몰래 게임하는 콘셉트와 맞물려 유머를 형성.

🏃 육상 게임 (멀리뛰기 / 달리기)
운동장 탑뷰 — 파워 게이지를 가득 채워 최고 기록을 달성하라!
타이밍 탑뷰 키 연타

🎮 핵심 메커닉

탑뷰 운동장에서 캐릭터(노란 캐릭터)가 달려나간다. 하단 좌측의 무지개색 파워 게이지를 키 입력으로 채워 최대 거리를 달성. 화면에 현재 기록 "11M" 등의 거리 수치 실시간 표시.

🏟️ 씬 구성

  • 초록색 잔디밭 탑뷰 그라운드
  • 흰 라인으로 구분된 트랙 레인
  • 우측에 학교 건물 일부 노출
  • 10M / 20M / 30M / 40M 거리 표시
  • 하단 좌측 무지개 파워 게이지

📊 UI 요소

  • 거리 카운터 — 현재 기록(M 단위)
  • 파워 게이지 — 7색 무지개 바
  • 레인 마커 — 거리 기준선

⚙️ 게임 루프

  • 준비: 출발선에 캐릭터 대기
  • 달리기: 좌우 키 빠르게 번갈아 입력
  • 도약: 적시에 점프키 → 멀리뛰기
  • 착지: 거리 기록 확정
  • 최고 기록 갱신 여부 판정
핵심 재미 요소: 무지개 게이지의 시각적 피드백이 강렬. 키보드 연타의 물리적 쾌감. "조금만 더 빠르게 했으면..."이라는 아쉬움이 재도전을 유도. 실제 육상 경기의 긴장감을 픽셀로 압축.

🏫 장대 오르기
학교 건물 외벽의 장대를 타고 꼭대기까지 — 중력을 이겨라!
리듬/타이밍 사이드뷰 키보드

🎮 핵심 메커닉

사이드뷰 배경의 학교 건물 외벽에 세워진 기다란 장대(사다리). 캐릭터가 장대를 타고 위로 올라가는 것이 목표. 타이밍에 맞춰 올라가는 리듬 게임 요소 포함.

🏗️ 씬 구성

  • 맑은 하늘 + 흰 구름 배경
  • 베이지색 학교 건물 외벽
  • 창문 격자가 배치된 건물
  • 하단 정원/화단 영역
  • 중앙에 수직으로 세워진 장대

⚙️ 게임 루프

  • 시작: 장대 하단에 캐릭터 위치
  • 입력: 방향키로 캐릭터 오르기
  • 도달: 장대 꼭대기 → 클리어
  • 실패: 체력/타이머 소진 or 추락
  • 기록: 최고 도달 높이 저장

🎯 도전 요소

  • 중력 — 손을 놓으면 미끄러짐
  • 체력 소모 — 오래 매달릴수록 감소
  • 타이머 — 제한 시간 존재
  • 장애물 — 구간별 방해 요소
핵심 재미 요소: 수직 상승의 쾌감과 추락의 공포가 공존. 건물 배경이 실제 학교 느낌을 강화하며 몰입도 상승. 간단한 컨트롤이지만 지구력과 타이밍이 요구되는 숙련 구조.

🪰 파리 잡기
모래사장을 날아다니는 파리들을 빠르게 잡아라 — 순발력 대결!
반응속도 탑뷰 마우스 클릭

🎮 핵심 메커닉

베이지/모래색 바닥 위에서 파리(벌레)들이 무작위로 이동한다. 플레이어는 마우스로 파리를 클릭해 잡으며 HighScore와 Win(승리 횟수)을 쌓는다. 파리의 이동 패턴이 불규칙적으로 예측 불가.

🏜️ 씬 구성

  • 베이지 모래색 사각형 경기장
  • 돌담 패턴 테두리 (경계선)
  • 파리 2~4마리 동시 등장
  • 보라+노란 파티클 효과 (잡을 때)
  • 간결한 UI (HighScore / Win)

📊 UI 요소

  • HighScore — 최고 연속 포획 수
  • Win — 라운드 승리 횟수
  • 파리 포획 시 파티클 이펙트

⚙️ 게임 루프

  • 시작: 파리 등장 (2~4마리)
  • 플레이: 마우스로 파리 클릭 포획
  • 클리어: 모든 파리 포획 → Win +1
  • 다음 라운드: 더 많은 파리, 빠른 이동
  • 실패: 타임아웃 → HighScore 비교
핵심 재미 요소: 실제 파리를 때려잡는 행위의 원초적 쾌감. 불규칙한 이동이 만드는 예측 불가능한 재미. 라운드마다 숫자가 늘어 점점 어려워지는 자연스러운 난이도 상승. "이번엔 다 잡을 수 있다!"는 욕구 자극.

강점

  • 직관적 조작 — 설명 불필요
  • 강한 클릭 쾌감
  • 다양한 파리 개수로 난이도 조절
  • 파티클 피드백이 시원함

개선 포인트

  • 파리 종류 다양화 필요
  • 특수 파리 (보너스/함정) 추가
  • 시간제한 UI 명확화
  • 연속 포획 콤보 시스템 검토

⚽ 탑뷰 축구 (BUG vs MARO)
실제 SBS 중계 화면 오버레이 — 픽셀 캐릭터들의 진짜 축구 대결!
스포츠 전략 탑뷰 키보드 이동

🎮 핵심 메커닉

탑뷰 축구장에서 BUG팀(파랑)과 MARO팀(주황)이 대결. SBS 스포츠 중계 화면처럼 스코어보드가 오버레이되어 현실감 극대화. 각 팀 픽셀 캐릭터들이 실제 포메이션으로 배치되어 이동·패스·슛.

🏟️ 씬 구성

  • 진초록 잔디 + 흰 라인 축구장
  • 센터서클 중앙 배치
  • SBS Sports LIVE HD 워터마크
  • 상단 스코어보드 오버레이
  • 팀별 색상 구분 (파랑/주황)

📊 UI 요소

  • 팀 스코어보드 — BUG 0:1 MARO
  • 경기 시간 — 분:초 표시
  • 팀 로고/뱃지 — 상단 좌측
  • 방송 스타일 중계 오버레이

⚙️ 게임 루프

  • 킥오프: 센터에서 공 배치
  • 이동: 방향키로 선수 컨트롤
  • 패스/슛: 버튼 입력
  • 전반/후반 구성 (제한시간)
  • 종료: 최종 스코어로 승패 결정
핵심 재미 요소: 실제 TV 스포츠 중계 화면 오버레이가 주는 유머와 현실감의 충돌. 픽셀 캐릭터들의 귀여운 비주얼과 정통 축구 규칙의 결합. 팀명(BUG/MARO)이 영상 세계관의 캐릭터들임을 암시해 스토리 연결.

⚽ 사이드뷰 1:1 축구
맑은 야외에서 1:1 대결 — 물리 기반 공을 컨트롤해 골을 넣어라!
스포츠 액션 사이드뷰 이동+점프

🎮 핵심 메커닉

사이드뷰 야외 축구장. 양 끝에 골대, 중앙에 공. 두 픽셀 캐릭터가 점프와 이동으로 공을 다루며 상대 골대에 넣는 게임. Cocos2D 물리엔진 기반 공 튕김이 예측 불가한 재미를 제공.

🌤️ 씬 구성

  • 밝은 하늘 + 흰 구름 배경
  • 초록 언덕 배경 + 나무
  • 양 끝 흰 골대 (좌/우)
  • 검은색 축구공 (물리 적용)
  • 양 팀 픽셀 캐릭터 (캐릭터 구분)

📊 UI 요소

  • 좌우 골대 스코어 (숫자 박스)
  • 점수 변경 시 깜빡임 효과
  • 골 효과음 및 이펙트

⚙️ 물리 설계

  • 중력: 공에 중력 상시 적용
  • 바운스: 땅/벽 반사 계수
  • 캐릭터 충돌: 공에 힘 전달
  • 점프 높이: 공 위 타격 가능
핵심 재미 요소: 물리 기반 공의 예측 불가 움직임이 만드는 우연성과 재미. 점프로 헤딩하는 공중전의 박진감. 단순한 룰(공 넣기)이지만 실력 차이가 극명하게 드러나는 구조. Slope Game 류의 물리 기반 캐주얼 스포츠 게임과 동일한 DNA.

🔦 어두운 탐험 맵
칠흑 같은 어둠 속 — 랜턴 하나로 학교의 비밀을 탐험하라!
어드벤처/RPG 탑뷰 키보드 이동

🎮 핵심 메커닉

전체 맵이 어두운 상태에서 캐릭터 주변 원형 시야(랜턴 효과)만 밝혀진다. 맵을 탐색하며 특정 구조물(건물, 가구)을 찾고 목적지에 도달하는 어드벤처 구조. 오버월드 이동 씬으로도 기능.

🌑 씬 구성

  • 전체 어두운 배경 (alpha 마스크)
  • 캐릭터 중심 원형 빛 범위
  • 어둠 속 건물/가구 실루엣
  • 특정 포인트 — 빛나는 상호작용 오브젝트
  • RPG 느낌의 탑뷰 타일맵

⚙️ 게임 루프

  • 입장: 어두운 맵에 캐릭터 배치
  • 탐색: 키보드로 이동, 시야 이동
  • 발견: 목표 오브젝트/NPC 조우
  • 상호작용: 버튼 입력 → 이벤트
  • 클리어: 탈출 or 목표 달성

🎯 설계 의도
